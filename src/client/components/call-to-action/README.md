# Call to action

A section that is used to persuade a user to some action.

## Usage

```html
<call-to-action
  :title="title"
  :body="body"
  :label="label"
  :url="url"
/>
```

## Props

| Prop | Type | Required |
| --- | --- | --- | --- |
| title | String | true
| body | String | false
| label | String | true
| url | String | true
