# Page Header
`isHomepage` is a boolean that toggles the homepage header variant,
`heading` defines which string is used as the `<h1>` inside the page-header

## Usage

```html
<page-header
  isHomepage
  context="We are De Voorhoede"
  tagline="How can we help?"
  header="context"
  :image="image"
>

</page-header>
```
